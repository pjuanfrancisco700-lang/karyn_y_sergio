# Sobrevive a la Isla — Frontend

Juego móvil listo para GitHub Pages.

## Archivos principales
- `index.html`
- `styles.css`
- `app.js`
- `config.js`
- `manifest.webmanifest`
- `sw.js`
- `assets/icon.svg`

## Configuración
Abre `config.js` y reemplaza:

```js
API_URL: "PEGA_AQUI_LA_URL_DE_TU_APPS_SCRIPT"
```

por la URL de la implementación web de Apps Script.

## Publicar en GitHub Pages
1. Sube todos los archivos a la raíz del repositorio.
2. Ve a **Settings > Pages**.
3. Selecciona la rama principal y carpeta `/root`.
4. Guarda.
5. Abre la URL publicada.

## Modo demostración
Mientras `API_URL` tenga el texto de ejemplo, la interfaz funcionará en modo visual de demostración.

## Recomendación
Abre la web desde Safari o Chrome y usa **Agregar a pantalla de inicio** para instalarla como PWA.
